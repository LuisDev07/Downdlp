# Downdlp

Application de escritorio pra la descarga de videos desde diversas plataformas.
utilizando la biblioteca `yt-dlp` para gestionar las descargas.
dandlole una interfaz gráfica amigable para facilitar su uso.

## Descripción
Descarga videos de plataformas como YouTube, Vimeo, Dailymotion, entre otras.

## Uso
Descararga la aplicacion .exe
